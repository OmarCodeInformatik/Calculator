# Calculator

Mein erstes richtiges JavaScript-Projekt: Ein klassischer Basic-Calculator mit cleanem Design.

Ein simpler Web-Taschenrechner für schnelle Rechnungen – gebaut mit purem HTML, CSS und JavaScript. Schwarz-weiß Look, modern und minimalistisch, mit wenig Farbe. Noch basic, aber ich bin richtig stolz, dass ich das alleine hingekriegt habe! 🧮✨

### Technologies
- `HTML`
- `CSS`
- `JavaScript`

Alles vanilla also kein Framework, kein Build-Tool, direkt im Browser und VS Code gecodet.

### Features
- Grundrechenarten: Addition (+), Subtraktion (−), Multiplikation (*), Division (/)
- Zahlen 0–9 und = zum Berechnen
- AC-Button zum eine Zahl oder Operation bei nur einem Klick zu löschen
- Hover-Effekte auf den Buttons (werden heller beim Drüberfahren)
- Zeigt "Infinity" bei Division durch 0 (JS-Standard)
- Responsive – funktioniert auch auf dem Handy (auch wenn's nicht perfekt ist)
- Kein Dezimalpunkt (.), keine Tastatur-Eingabe (nur Maus/Klick), keine Extra-Features wie %, Wurzel oder Dark Mode

### The Process
Ich habe das mit einem Tutorial gebaut – als mein allererstes echtes Projekt wollte ich einfach mal JavaScript richtig lernen und sehen, ob ich so was hinbekomme. Am schwierigsten war die Rechenlogik (Operatoren richtig verketten und berechnen), auch wenn es eigentlich "einfach" war. Dabei habe ich mega viel gelernt: JavaScript Basics, Events mit `onclick`, `:hover` in CSS, DOM-Manipulation und wie man Bugs debuggt.  

Noch basic, aber guter Start – nächstes Mal wird's besser (z. B. mit Dezimalen, Tastatur-Support und mehr Features). Aber ich finde es richtig toll, dass ich überhaupt sowas gemacht habe!
